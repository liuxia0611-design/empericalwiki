# Runtime Support Files

> On-demand reference for graph-derived files plus the non-page runtime files `index.md` and `log.md`.

## Graph Files

| File | Content | Generated by |
|------|---------|-------------|
| `edges.jsonl` | semantic relationships: paper-paper, paper-variable, paper-model, claim/experiment/idea/provenance edges | `python3 tools/research_wiki.py add-edge` |
| `citations.jsonl` | bibliographic paper citations (`type: cites`) | `python3 tools/research_wiki.py add-citation` |
| `context_brief.md` | compressed context: claims + gaps + failed ideas + papers + edges (≤8000 chars) | `python3 tools/research_wiki.py rebuild-context-brief` |
| `open_questions.md` | open questions: under-supported claims + open questions from papers/topics | `python3 tools/research_wiki.py rebuild-open-questions` |

Semantic edge format: `{"from": "node_id", "to": "node_id", "type": "edge_type", "evidence": "...", "confidence": "high|medium|low", "date": "..."}`

Citation format: `{"from": "papers/citing", "to": "papers/cited", "type": "cites", "source": "semantic_scholar|parsed_bib|manual", "date": "..."}`
